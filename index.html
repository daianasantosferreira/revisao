Conhecimento!

Widgets

Widgets são componentes de interface que exibem informações e permitem interações rápidas diretamente na tela inicial de dispositivos, como smartphones e desktops. Eles facilitam o acesso a dados e funcionalidades sem abrir aplicativos completos.

VisualG

VisualG é uma ferramenta educacional usada para ensinar lógica de programação. Com uma interface simples, permite criar algoritmos visuais e textuais, facilitando o aprendizado de conceitos básicos de programação para iniciantes. Ele foi importante para o aprimoramento do nosso conhecimento de JavaScript.

Expo

Expo é uma ferramenta e plataforma para criar aplicações móveis usando React Native. Ela simplifica o desenvolvimento ao fornecer ferramentas e bibliotecas prontas, permitindo que os desenvolvedores se concentrem na criação de funcionalidades. Foi utilizado para iniciar nosso projeto.

SQL

SQL é uma linguagem de programação para gerenciar e manipular bancos de dados relacionais. Permite realizar consultas, inserir, atualizar e excluir dados, além de criar e modificar a estrutura dos bancos. Usamos para criar tabelas e aprimorar a filtragem de informações com base em dados específicos.

Arduino

Arduino é uma plataforma de prototipagem eletrônica open-source baseada em hardware e software fáceis de usar. Utilizamos para aprender sobre sensores, LEDs, montagem de um sistema básico de semáforo e displays numéricos.

Node-RED

Node-RED é uma ferramenta de programação baseada em fluxo para conectar dispositivos de hardware, APIs e serviços online. Permite criar aplicações de integração facilmente com uma interface gráfica intuitiva. Usamos para criar funções no Arduino físico através de um complemento.

Ubuntu Server

Ubuntu Server é uma distribuição Linux voltada para servidores. Conhecida por sua estabilidade, segurança e grande comunidade de suporte, foi utilizada para criar uma máquina virtual e montar nosso app com o Expo.

Prompt

Prompt, ou linha de comando, é uma interface de texto para digitar comandos e executar no sistema operacional. Foi interligado ao Ubuntu Server para facilitar a colagem de elementos no código, sendo uma ferramenta poderosa para administrar sistemas, automatizar tarefas e executar scripts.

Banco de dados

Um banco de dados é uma coleção organizada de informações sobre um mesmo assunto, normalmente armazenadas eletronicamente em um sistema de computador.
